# Project Brief - Patee E-commerce Website

## Project Overview
**Patee** is a Vietnamese family clothing e-commerce website with the tagline "Wear The Love". The project focuses on providing fashionable clothing for all family members - from grandparents to children.

## Core Requirements

### Business Goals
- Create an online store for family clothing
- Target Vietnamese market with Vietnamese language interface
- Provide products for all family demographics (Ông/Bà, Cha/Mẹ, Con Trai/Con Gái)
- Implement promotional features (sales, new arrivals)

### Functional Requirements
1. **Product Display**
   - Dynamic product loading from JSON data
   - Product categorization by family member
   - Color variant selection
   - Price display with sale pricing

2. **User Interface**
   - Responsive design using Bootstrap 5.3.8
   - Modern, clean aesthetic with coral pink theme
   - Interactive product cards with hover effects
   - Countdown timer for sales promotions

3. **Navigation**
   - Multi-level navigation with dropdowns
   - Category-based product filtering
   - Search functionality
   - Shopping cart integration

4. **Content Management**
   - JSON-based product data management
   - Separate datasets for different product types
   - Component-based architecture for reusability

### Technical Constraints
- Frontend-only implementation (no backend)
- JSON file-based data storage
- Bootstrap framework dependency
- Vietnamese language support
- Mobile-responsive design

### Success Criteria
- Functional product catalog with dynamic loading
- Smooth user experience across devices
- Effective promotional sections (sales, new arrivals)
- Clean, maintainable code structure
- Fast loading times with optimized assets

## Project Scope
This is a frontend e-commerce website focusing on family clothing retail. The scope includes:
- Homepage with promotional sections
- Product catalog with filtering
- Shopping cart functionality
- Responsive design implementation
- Vietnamese localization

## Out of Scope
- Backend API development
- User authentication system
- Payment processing
- Order management system
- Inventory management
- Multi-language support beyond Vietnamese
