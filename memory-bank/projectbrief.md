# Project Brief

## Project Overview
**Patee E-commerce Website** - A Vietnamese family fashion brand specializing in matching family outfits, couples' clothing, mother-child sets, and children's apparel.

## Core Requirements
- **Platform**: Static website with vanilla HTML, CSS, JavaScript
- **Framework**: Bootstrap 5.3.8 for responsive design
- **Data**: JSON-based product catalog with localStorage cart
- **UI**: Vietnamese language interface with Bootstrap Icons
- **Cart**: Slide-in panel cart system (recently implemented)
- **Components**: Dynamic header/footer loading system

## Key Features
1. **Product Catalog**: 25+ items across 5 categories (Gia đình, Mẹ và bé, Bé trai, Bé gái, Đồ đôi)
2. **Dynamic Loading**: Header/footer loaded via JavaScript for consistency
3. **Shopping Cart**: Full-featured cart with add/remove/quantity adjustment
4. **Responsive Design**: Mobile-first approach using Bootstrap
5. **Product Categories**: Individual pages for each clothing category
6. **User Authentication**: Login, registration, password recovery pages

## Technical Constraints
- **No Backend**: Pure frontend with static JSON data
- **CORS Dependencies**: Requires local web server for JSON fetch calls
- **No Build Process**: Direct HTML/CSS/JS development
- **LocalStorage**: Cart data persistence in browser storage

## Success Metrics
- User-friendly shopping experience
- Mobile responsiveness across all devices
- Fast loading times with efficient component loading
- Intuitive cart management with slide-in interface
- Consistent Vietnamese language UX throughout

## Project Scope
- Complete e-commerce frontend with product browsing
- Functional cart system with checkout flow
- Category-based product organization
- User authentication pages (frontend only)
- Admin-free operation with static data management
